# tmux-web/static/js/tasks/crud.js

Tasks CRUD + optimistic UI. Экспорты: getIssueIndex, applyOptimisticPatch, rollbackIssue, createTask (POST /api/tasks), updateTask (PATCH /api/tasks/:id), taskOriginById, closeTask (DELETE /api/tasks/:id?reason — закрывает issue, не удаляет), reopenTask (POST /api/tasks/:id/reopen), promoteTodo (POST /api/todos/:id/promote). НОВОЕ (forge-caz3): purgeTask(id) — POST /api/tasks/:id/purge для физического удаления (br delete --hard --force). deleteTodoLocal(id) — DELETE /api/todos/:id для bulk-clean колонки TODO. cleanColumn(status, ids) — batch-операция: для closed вызывает purgeTask, для todo — deleteTodoLocal, для остальных — closeTask(id,'clean-column'). Возвращает {ok, fail}. Все три новые функции с optimistic remove + rollback при ошибке.
