# tmux-web/static/js/tasks/crud.js

Tasks CRUD + optimistic UI. Экспорты: getIssueIndex, applyOptimisticPatch, rollbackIssue, createTask, updateTask, taskOriginById, closeTask(id, reason, opts?), reopenTask, promoteTodo. ИЗМЕНЕНИЯ (forge-caz3 фикс): purgeTask(id, opts?) — POST /api/tasks/:id/purge (br delete --hard --force); deleteTodoLocal(id, opts?) — DELETE /api/todos/:id; cleanColumn(status, ids). opts.silent=true подавляет window.alert при ошибках (используется при batch-вызовах из cleanColumn чтобы не залить пользователя N alert-окнами). cleanColumn теперь использует Promise.allSettled с CONCURRENCY=8 параллельных запросов на chunk — N задач удаляются за ceil(N/8) round-trip'ов вместо N последовательных. Backend handler purge_task идемпотентен: ISSUE_NOT_FOUND трактуется как успех.
