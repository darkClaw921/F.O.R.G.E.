# tmux-web/static/app.js::windowBar

UI-таб-бар окон tmux над терминалом (#window-bar в index.html, между #tab-bar и #terminal).\n\nDOM:  (.window-bar),  (.window-tabs scroll-flex контейнер),  (+).\n\nState:\n- state.currentWindows: WindowInfo[] — последний snapshot окон активной сессии.\n- state.windowsPollTimer: setInterval handle для poll каждые 2 секунды.\n\nЖизненный цикл:\n- ws.onopen → startWindowsPolling() — первый fetch + setInterval 2000ms.\n- disconnectWs() → stopWindowsPolling() — clearInterval + currentWindows=[] + render.\n- switchTab() обновляет hidden у .\n\nФункции:\n- fetchWindows() — GET /api/sessions/:name/windows через apiFetch (origin-aware). При 400/404 → очищает state.currentWindows (сессии уже нет).\n- renderWindowBar() — рендерит таб для каждого WindowInfo. Активный получает .active. Содержимое: span.window-tab-idx (индекс), span.window-tab-name (имя), button.window-tab-close (×). Клик → selectWindow, double-click → renameWindow, клик по × → killWindow с confirm.\n- createWindow() — prompt имени (пусто → дефолт tmux) → POST /api/sessions/:name/windows.\n- selectWindow(index) — POST /api/sessions/:name/windows/:index/select.\n- killWindow(index, name) — confirm + DELETE.\n- renameWindow(index, oldName) — prompt + PATCH.\n\nКнопка  привязана к createWindow в bootstrap.
