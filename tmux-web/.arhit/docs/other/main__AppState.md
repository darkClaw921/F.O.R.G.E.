# main::AppState

Главное shared state приложения в src/main.rs (~line 54). Поля: projects: Arc<RwLock<projects::ProjectStore>> — список tmux-проектов и активный проект; tasks: Arc<...> — task tracker; pty: Arc<...> — PTY-сессии для веб-терминалов; attention: Arc<attention::AttentionState> (Phase 2.1) — разделяемое состояние флагов 'нужно внимание' для tmux-сессий. AttentionState заполняется фоновым watcher'ом attention::watcher_loop, спавнится в main() сразу после создания app_state через tokio::spawn(attention::watcher_loop(app_state.projects.clone(), app_state.attention.clone())) — передаются именно компоненты (Arc-клоны), а не AppState целиком, чтобы избежать циклической зависимости attention->main->attention. Хендлер /api/sessions читает attention.snapshot() и склеивает с tmux::list_sessions в Vec<SessionDto>. AppState клонируется (Clone) для axum::extract::State — все Arc-поля делятся cheaply. Файл: src/main.rs.
