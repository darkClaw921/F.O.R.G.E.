# session-item-needs-attention

CSS-класс в static/style.css (~line 197), визуализирующий флаг needs_attention для tmux-сессии в сайдбаре. Применяется когда Claude в сессии ждёт ввода (permission prompt и т.п.). Три селектора: 1) .session-item.needs-attention — базовый стиль: тёмно-оранжевый фон #3a2010, оранжевая левая граница #ff8a3d. 2) .session-item.needs-attention .session-name — имя сессии становится оранжевым (#ff8a3d) и жирным (font-weight 600) для усиления визуального сигнала. 3) .session-item.needs-attention.active — комбинация с активной вкладкой: более насыщенный тёмно-оранжевый фон #4a2814, граница остаётся синей (наследуется от .active, так как .active объявлен раньше). Расположен после блока .session-item.active в каскаде, что важно для корректного приоритета при комбинации классов. Класс навешивается из renderSidebar() в app.js.
