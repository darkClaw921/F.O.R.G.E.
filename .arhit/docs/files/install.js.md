# install.js

PWA install UI (Фаза 5, tmux-web/static/js/pwa/install.js). Лениво импортируется из bootstrap.js ТОЛЬКО при enabled (opt-in). initInstall() (идемпотентна через window.__FORGE_PWA_INSTALL_INIT): на Android/desktop ловит beforeinstallprompt (preventDefault + сохраняет deferredPrompt) и показывает кастомный чип .pwa-install-chip '#pwa-install-chip' в #settings-bar; клик -> deferredPrompt.prompt() + await userChoice (event одноразовый); appinstalled -> скрывает чип. На iOS (isIos(): iphone/ipad/ipod UA + iPadOS-детект через macintosh UA & maxTouchPoints>1) и не standalone -> одноразовая текстовая подсказка .pwa-ios-hint 'Поделиться -> На экран Домой', dismiss сохраняется в localStorage key 'forge-pwa-ios-install-dismissed'. Экспорты: initInstall, isStandalone (display-mode standalone/minimal-ui или navigator.standalone), isIos. Стили в css/pwa.css (.pwa-install-chip уже была; .pwa-ios-hint добавлена в Фазе 5). Самодостаточен — не зависит от app-модулей.
