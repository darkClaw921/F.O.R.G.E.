# plugins/echo/src/db/repo/autonomous.rs

CRUD для autonomous_tasks и task_runs. AutonomousTask{id (UUIDv4), name, prompt_template, interval_seconds, model, enabled: bool, project_id (Option soft-FK), last_run_at, next_run_at, created_at}. TaskRun{id, task_id (FK cascade), started_at, finished_at: Option, status (running/success/error/cancelled), result_message_id, tokens_in/out, error}. TaskPatch — partial update через COALESCE(?, col). API: create_task (next_run_at = now+interval), list_tasks (enabled_only), get_task, update_task, delete_task, set_next_run, find_due(now) — для scheduler Phase 4, использует idx_autonomous_enabled_next; insert_run (зеркалит started_at в task.last_run_at), finish_run, list_runs. 4 unit-теста: CRUD, find_due+disable, run lifecycle, cascade delete.
