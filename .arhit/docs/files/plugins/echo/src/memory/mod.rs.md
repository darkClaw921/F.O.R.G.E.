# plugins/echo/src/memory/mod.rs

Memory automation module for Echo plugin. Provides summarize_day(state, host, day, project_id) — собирает messages за день UTC + capture-pane, отправляет в ClaudeRunner с мета-prompt'ом 'Summarize key facts and decisions from this day. Output concise markdown (≤ 1500 tokens). Focus on what changed.', upsert'ит в memories под scope global_day (project_id=None) или project_day (project_id=Some). При отсутствии данных за день пишет content='No activity', source='auto'. summarize_project(state, host, project_id) — собирает N=20 последних project_day memories + 200 messages за неделю + capture-pane проекта, мета-prompt 'Build/update a stable project memory: architecture, decisions, current focus.', upsert'ит под scope='project'. Constants: MAX_DAY_MESSAGES=200, MAX_PROJECT_DAY_MEMORIES=20, MESSAGE_SNIPPET_CAP=800. Helpers: day_bounds_utc, snippet, collect_day_messages, collect_week_messages, collect_pane_snapshot.
