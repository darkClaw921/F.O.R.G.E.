# plugins/echo/src/routes/daily_reports.rs

REST-роутер фичи «Сводка дня» (forge-echo). Монтируется в build_router под /api/echo/daily-reports. Эндпоинты: GET /api/echo/daily-reports — список отчётов (daily_reports::list, ответ {items:[...]}), отсортировано day DESC; GET /api/echo/daily-reports/:day — отчёт за конкретный день (валидирует YYYY-MM-DD → 400 на битый формат, 404 если нет записи); POST /api/echo/daily-reports/generate — body {day?: 'YYYY-MM-DD'} (без day → сегодня по local-времени), генерит через daily_report::generate_report с source='manual', таймаут ~90с, 200 c DTO отчёта на успех, 400 на невалидный day. Все хост-вызовы идут через state.host (Arc<dyn HostApi>). Тесты (mock-CLI + in-memory Db): generate 200/400, get_by_day 404/400, list+get roundtrip, default day = today local.
