# next-step-popup.js

Интерактивный hover-попап фичи «Следующий шаг» (tmux-web/static/js/sessions/next-step-popup.js). Singleton .next-step-popup на body (position:fixed), показывается при наведении на .session-item.has-next-step через делегированный mouseover на document (переживает пересборку сайдбара каждые 3с). Отличие от ui/tooltip.js: попап ИНТЕРАКТИВНЫЙ — удерживается пока курсор над ним (mouseenter попапа отменяет отложенное скрытие hideTimer/HIDE_DELAY_MS=160ms). Содержит: textarea с текстом предложения (state.nextSteps[name].content), кнопку «Отправить в терминал» → sendNextStep(session, text актуальный из textarea) → POST /api/echo/next-steps/:session/send; поле коррекции + кнопку «Сохранить» → feedbackNextStep(session, correction) → POST .../feedback. После действия clearLocal() оптимистично удаляет предложение из state.nextSteps + renderSidebar (WS подтвердит). Экспорт initNextStepPopup() (идемпотентен) вызывается из core/bootstrap.js. Зависит от: core/state.js (nextSteps), sidebar/sidebar.js (renderSidebar), echo/api.js (sendNextStep/feedbackNextStep), echo/notifications.js (notify).
