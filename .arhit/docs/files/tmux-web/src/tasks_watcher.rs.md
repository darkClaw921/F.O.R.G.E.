# tmux-web/src/tasks_watcher.rs

Фоновый file-watcher для .beads/issues.jsonl активного пути. После Phase 4 (remove-projects-concept) переименован project_path → active_path, привязка к ProjectStore убрана. Использует notify crate (RecommendedWatcher) с tail-debounce 200ms. run_watcher(active_path_rx, tasks_tx) — единственный публичный entry-point. Outer-loop пересоздаёт watcher при смене active_path (sender в AppState — extension-point). Inner-loop: snapshot baseline → notify event → 200ms debounce → diff_issues → broadcast TaskEvent. find_beads_dir() ищет .beads/ начиная с указанного пути вверх по родительским каталогам (зеркалит логику ). relevant_event() — фильтр fs-событий (только issues.jsonl и его .tmp). DEBOUNCE_MS = 200.
