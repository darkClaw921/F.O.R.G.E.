# tmux-web/src/main.rs

Главный entry-point devforge: axum-сервер для tmux web-viewer + REST/WS API. После Phase 4 (remove-projects-concept) поле AppState.projects удалено, маршруты /api/projects/* сняты; концепция «проектов» больше не существует. AppState ключевые поля: tasks_tx (broadcast TaskEvent для notifier и WS-tasks), active_path_tx (watch::Sender<PathBuf> — последний активный путь, инициализирован cwd процесса), attention (Arc<AttentionState>), notify (NotifyHandle), todos (TodoStore), todos_tx, themes/themes_dir, remote_mode, auth_token, remotes, http (reqwest::Client), user_settings, notifier_config. Endpoints: Sessions API (GET/POST/DELETE/PATCH /api/sessions...), Tasks API (GET/POST/PATCH/DELETE /api/tasks...), Todos API, Themes API, User-settings API, Notifier-config API, WS routes (/ws/attach, /ws/lazygit, /ws/lazydocker, /ws/telescope, /ws/tasks, /ws/todos). Remote-mode: /api/remote-servers/* регистрируются только при флаге remote_mode. resolve_folder() — единственная сохранённая helper-функция из удалённого projects-блока (вычисляет folder_id/folder_label для sidebar-группировки). SessionDto чистый от project_id/project_name/tmux_prefix полей.
