# tmux-web/static/css/tui-tab.css

Generic TUI tab — обобщение .git-* для повторного использования через createTuiTab() factory: #docker (lazydocker), #telescope (find), и других TUI-приложений. .tui-pane, .tui-* классы аналогичны .git-*. Включает также Channel-switcher bar (.tui-channel-bar) для Find-вкладки: переключатель Files/Content/Dirs/Git log — .tui-channel-btn с :hover/.active. 221 строка (2352-2572 оригинала).
