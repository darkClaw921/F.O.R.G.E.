# tmux-web/static/css/echo.css

Echo plugin CSS: layout grid (300px sidebar + 1fr main), conversations list с active state, chat messages (user right-aligned with accent tint, assistant left-aligned), markdown rendering (.echo-md-pre/.echo-md-code/.echo-md-h/.echo-md-ul), input area, model picker, autonomous list, memory viewer (3 tabs), modals (echo-modal-overlay/.echo-modal/.echo-modal-form/.echo-field). Все цвета через CSS variables — реагируют на theme switch.
