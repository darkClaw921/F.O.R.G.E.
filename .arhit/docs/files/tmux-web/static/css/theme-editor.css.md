# tmux-web/static/css/theme-editor.css

Custom theme editor модал — расширенная форма создания пользовательской темы: .modal-card.theme-editor-modal (фиксированная ширина 720px, max-width 92vw), grid из color-picker'ов для каждой CSS-переменной (--bg, --fg, ANSI палитра и т.д.), background+foreground шире (доминирующие), live preview блок с примером терминала. 376 строк. Используется при создании/редактировании пользовательских тем (Phase wk7).
