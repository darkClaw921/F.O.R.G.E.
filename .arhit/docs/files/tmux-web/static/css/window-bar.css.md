# tmux-web/static/css/window-bar.css

Стили нижнего бара с tmux window tabs внутри активной сессии: #window-bar (контейнер 0 0 28px), .window-tab (вкладка окна с pseudo-classes :hover/.active/.dirty/.zoomed), кнопки add/close/rename, индикатор unread. 98 строк. Соответствует логике renderWindowBar() из tmux-web/static/js/sessions/windows.js.
