# tmux-web/static/js/settings/notifications-tab.js

Notifications-вкладка settings modal. После Phase 5 заменяет project-specific нотификации на единый глобальный конфиг. Экспорты: fetchNotifierConfig() — GET /api/notifier-config, возвращает NotifierConfig (template, delay_minutes, wait_previous, session); saveNotifierConfig(patch) — PATCH /api/notifier-config с частичным обновлением, возвращает {ok, config?, error?}; buildNotificationsForm(config, onSaved) — строит DOM-форму с полями template (textarea), delay_minutes (number), wait_previous (checkbox), session (text, пустая строка ⇒ сброс в None на бэкенде). Backend: ~/.config/forge/notifier.json через tmux-web/src/notifier_config.rs.
