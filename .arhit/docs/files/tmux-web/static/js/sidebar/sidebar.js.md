# tmux-web/static/js/sidebar/sidebar.js

Frontend рендеринг сайдбара со списком tmux-сессий. После Phase 5 нет project-уровня — только folder-группы. Экспорты: renderSidebar — основная точка входа, в remote-mode делегирует renderSidebarWithOrigin (origin-секции local + remote-серверы); renderSidebarWithOrigin — двухуровневая структура origin → folder-группы; renderOriginSection(originKey, label, dotKind, sessions, opts) — секция одного origin с collapsable header (caret + dot + label + offline-badge или sess count), отрисовывает folder-группы или Loading/Offline placeholder. Внутренний renderFolderGroups(sessions, opts) — общая логика группировки sessions по folder_id/folder_label с заголовками; orphan-сессии (без folder_id) рендерятся плоско если folder-групп нет, иначе под заголовком 'All sessions'. Без projects, без projectFilter.
