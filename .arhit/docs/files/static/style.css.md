# static/style.css

Стили tmux-web (dark theme). Layout: #layout (flex-row, 100vh) → #sidebar (240px) + #main (flex-col). #main теперь flex-column (Phase 6.A): #tab-bar (36px высота, кнопки .tab-btn с .active=#2a7fff) → #terminal (flex:1; min-height:0; position:relative — xterm требует relative parent) и #tasks (flex:1; hidden до switchTab). Tasks board: #tasks-board grid с repeat(auto-fit, minmax(240px, 1fr)). Колонки .kanban-col имеют sticky-header (.kanban-col-header) и скроллящееся .kanban-col-body. Карточки .kanban-card с цветным border-left-color по data-priority (0=#e84d4d, 1=#e89344, 2=#2a7fff, 3/4=#555); .p-pill с тем же цветом, .type-tag и .label-tag в нейтральных. Sidebar/session-list стили — без изменений. Убран старый position:absolute/inset:0 с #terminal — теперь flex layout. .placeholder остаётся position:absolute поверх #terminal.
