# static/app.js

Frontend tmux-web (vanilla JS, IIFE). Состояние в state: term/fitAddon/webLinksAddon (xterm.js), ws/currentSession/sessions/pollTimer (sessions-polling 3s + WS /ws/attach), encoder, lastResizeKey. Phase 6.A добавил activeTab ('terminal'|'tasks'), tasksPollTimer, tasksData. Функции терминала: initTerminal/sendResize/connectWs/disconnectWs/switchSession/scheduleResizeFromTerm/openSession + sidebar (fetchSessions, renderSidebar, startPolling/stopPolling, createSessionPrompt, killSession). Phase 6.A добавил функции Tasks-таба: switchTab(name) — toggle hidden и .active классы, refit xterm + scheduleResizeFromTerm при возврате на terminal, при переходе на tasks — fetchTasks (если data null) и startTasksPolling (5s); fetchTasks — GET /api/tasks с graceful degradation; renderTasks — группировка по status (TASK_COLUMNS=open,in_progress,blocked,deferred,draft,closed), sort priority asc → updated_at desc, closed обрезается до 20; renderColumn/renderCard формируют DOM с data-priority для CSS-цветов и P-pill/type-tag/labels (max 3 + +N); setTasksStatus — текст в #tasks-status. Bootstrap: листенеры на // + visibilitychange/beforeunload теперь стопают tasksPollTimer. tasks-new — alert-заглушка (Phase 6.C). compareIssues: priority asc, updated_at desc, отсутствующие поля → 5 / ''.
