# renderHome

renderHome (js/home/home.js) — async-функция рендера главной страницы 'Недавние сессии'. Загружает историю через apiFetch('/api/sessions/history') (GET, массив HistorySession {name,path,folder_label,windows:[{index,name}],first_seen,last_seen}), очищает грид #home-cards и строит карточку на каждую запись через buildHomeCard (document.createElement, XSS-safe — без innerHTML с пользовательскими данными). Карточка содержит: имя сессии, folder_label, path, количество и список имён окон, кнопки '▶ Запустить' (restoreSession) и '✕' (deleteHistory). При пустой истории показывает #home-empty ('История пуста') и скрывает кнопку #home-restore-all. При первом вызове один раз навешивает listener на #home-restore-all → restoreAll (флаг restoreAllBound защищает от дублей). Связанные функции модуля: restoreSession(name,path) — POST /api/sessions/history/restore, обрабатывает 409 (имя занято), при успехе fetchSessions()+openSession(name); restoreAll() — POST /api/sessions/history/restore-all + fetchSessions(); deleteHistory(name,path) — DELETE /api/sessions/history {name,path} + renderHome(); showHome(show) — переключает display #home (flex/none), при show=true вызывает showPlaceholder(false). Вызывается из sessions.js::fetchSessions (через syncHomeVisibility) когда нет активных сессий и нет currentSession. Зависимости: apiFetch (core/api.js), fetchSessions/openSession (sessions/sessions.js), showPlaceholder (terminal/xterm.js), DOM $home/$homeCards/$homeRestoreAll/$homeEmpty (core/dom.js).
