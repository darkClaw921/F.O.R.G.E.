# renderDailySummaryTab

Фабрика вкладки «Сводка дня» в settings modal (tmux-web/static/js/settings/daily-summary-tab.js). Сигнатура: renderDailySummaryTab(container, { onClose }). Наполняет переданный контейнер DOM-узлами: fieldset .notify-fieldset с легендой/подсказкой и двумя кнопками в .notify-actions. Кнопка .ds-generate «Сгенерировать сводку за сегодня» вызывает generateDailyReport() из echo/api.js без day (= сегодня), показывает статус в .ds-status со сменой классов pending/ok/error (паттерн как «Test connection» в remotes-tab); на время запроса кнопка disabled. Кнопка .ds-open «Открыть страницу» вызывает onClose() (закрытие модалки) и window.ForgeApp.showDailySummary() — точка интеграции из Phase 4. Зависит от generateDailyReport (echo/api.js) и глобального window.ForgeApp. Вкладка не хранит user-settings, рендерится лениво один раз при первом показе.
