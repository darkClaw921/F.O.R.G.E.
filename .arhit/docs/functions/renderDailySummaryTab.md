# renderDailySummaryTab

tmux-web/static/js/settings/daily-summary-tab.js: вкладка настроек «Сводка дня». Кнопка «Сгенерировать сводку за сегодня» запускает generateDailyReport() в ФОНЕ — промис не блокирует UI (модалку можно закрыть), по завершении показывается glass-уведомление справа (notify({glass:true})): успех «Сводка дня готова» или ошибка. Флаг модуля _bgGenerating защищает от параллельных запусков. Кнопка «Открыть страницу» → window.ForgeApp.showDailySummary(). Секция «Промпты генерации»: два textarea (report/suggest) + Сохранить + по кнопке «↺ дефолт», через getDailyReportPrompts/saveDailyReportPrompts.
