# openTodoEditModal

tmux-web/static/js/tasks/modals.js — модалка редактирования TODO. Поля: Title (#td-title), Description (#td-desc), чекбокс план-мода (#td-plan-mode), чекбокс авто-промоута (#td-auto-promote, checked если todo.auto_promote, подпись 'Авто-запуск после предыдущей задачи в очереди'), поле Promote target session (#td-session). Save-обработчик собирает patch только из изменённых полей: title, description, plan_mode (newPlanMode !== !!todo.plan_mode), auto_promote (newAuto = .checked, добавляется если newAuto !== !!todo.auto_promote). Если patch пуст — просто закрывает модалку. Иначе шлёт PATCH /api/todos/:id через apiFetch(url, opts, dtoOrigin(todo)); сервер рассылает WS Upsert. Кнопки Delete (DELETE /api/todos/:id с опциональным confirm) и Promote (вызывает promoteTodo с введённой сессией).
