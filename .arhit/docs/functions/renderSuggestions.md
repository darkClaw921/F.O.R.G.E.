# renderSuggestions

Рендер блока «Предлагаемые задачи по проектам» под markdown-сводкой (tmux-web/static/js/daily-summary/daily-summary.js). Карточки получают class='kanban-card suggestion-card' + data-status='todo' → фирменный TODO-вид (фиолетовый бордер, стилизованные .desc/.p-pill). Внутри: чекбокс-индикатор .suggestion-check (заливается галочкой при выборе) + .suggestion-body (.title, .desc до 140 символов, .meta-row с .p-pill 'P{priority}' и type-tag 'todo'). Клик тоглит class 'selected' и aria-pressed, обновляет Set выбранных и disabled кнопки. Кнопка «Добавить выбранные в TODO» последовательно вызывает createTodoForPath(project_path,...) для выбранных; успешные → class 'added' + пометка '✓ добавлено', снимаются с выбора. Под заголовком — подсказка .daily-summary-suggestions-hint. XSS-safe (createElement/textContent).
