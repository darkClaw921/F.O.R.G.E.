# renderGantt

Основной рендер гант-диаграммы вкладки Tasks (gantt.js). Чистит $ganttCanvas, фильтрует issues по статусу in_progress|closed с валидной created_at, вычисляет домен [t0,t1] через ganttWindow и перерисовывает ось/строки/коммиты. Раскладка прокрутки: ось .gantt-axis аппендится прямым потомком #gantt-canvas (sticky к верху скролл-контейнера), затем создаётся контейнер .gantt-rows (position:relative) — в него аппендятся все .gantt-row, заглушка .gantt-empty (если нет видимых задач) и overlay коммитов. Overlay внутри .gantt-rows (top/bottom:0) покрывает полную высоту всех строк, поэтому вертикальные черты коммитов доходят до самого низа при прокрутке #gantt-canvas. Пустые состояния: t0===null или вырожденный домен -> renderEmpty(canvas); visible.length===0 -> ось + .gantt-empty + renderCommits(rowsEl,...) внутри .gantt-rows.
