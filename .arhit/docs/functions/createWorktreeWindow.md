# createWorktreeWindow

Frontend-обработчик кнопки ⑂ (#window-new-worktree). Файл: tmux-web/static/js/sessions/windows.js. Экспортируемая async-функция без аргументов и без window.prompt (авто-режим, один клик): берёт state.currentSession, шлёт POST /api/sessions/:name/windows/worktree через apiFetch с state.attachWsOrigin. При !ok && status!==201 показывает alert с текстом ответа (например 400 на не-git-сессии), иначе вызывает fetchWindows() для обновления полосы окон. Привязка клика — bootstrap.js: $windowNewWorktreeBtn.addEventListener('click', createWorktreeWindow). Парная логика удаления — в killWindow(index,name): окна с именем-префиксом 'wt:' закрываются через DELETE /api/sessions/:name/windows/:index/worktree с усиленным confirm (удаление git worktree; коммиты ветки forge/ сохраняются), обычные окна — прежним DELETE .../windows/:index.
