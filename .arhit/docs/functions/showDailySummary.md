# showDailySummary

Точка входа вью «Сводка дня» (tmux-web/static/js/daily-summary/daily-summary.js). showDailySummary(day?) показывает контейнер #daily-summary, скрывает #home и placeholder (по образцу home.js::showHome), выставляет _currentDay (по умолчанию сегодня) и грузит сводку через loadCurrent. loadCurrent вызывает getDailyReport(day): при 404 показывает пустое состояние с кнопкой «Сгенерировать», иначе рендерит markdown через renderMarkdownInto (core/markdown.js). Навигация по датам: prev/today/next (next блокируется, если день >= сегодня — запрет будущего). generateCurrent вызывает generateDailyReport и перерендеривает; кнопки блокируются на время запроса (setBusy). Также экспортирует hideDailySummary(). Зависимости: echo/api.js (getDailyReport/generateDailyReport), core/markdown.js (renderMarkdownInto), terminal/xterm.js (showPlaceholder), core/dom.js ($dailySummary*). Вью открывается извне через window.ForgeApp.showDailySummary (Phase 5 — кнопка в настройках).
