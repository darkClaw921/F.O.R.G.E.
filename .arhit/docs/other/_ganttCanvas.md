# $ganttCanvas

DOM-ссылка на #gantt-canvas (tmux-web/static/js/core/dom.js) — relative-контейнер, в котором gantt.js (Phase 3) абсолютно позиционирует .gantt-row/.gantt-bar (полосы задач по статусу), .gantt-commit (вертикальные черты git-коммитов) и .gantt-axis/.gantt-tick (ось дней). Стили в tmux-web/static/css/tasks.css раздел 'Gantt timeline'.
