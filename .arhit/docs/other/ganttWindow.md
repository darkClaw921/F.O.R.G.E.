# ganttWindow

Хелпер домена гант-диаграммы (tmux-web/static/js/tasks/gantt.js, exported). Вход rows (отфильтрованные строки задач {start} в мс, нужны только для 'all'). Возврат {t0,t1,since,until}: t0/t1 — границы домена в МИЛЛИСЕКУНДАХ для рендера; since/until — те же в СЕКУНДАХ Unix для GET /api/git/commits, null = не ограничивать. Семантика по state.ganttRange: 'today' — t0=начало сегодняшних локальных суток (setHours(0,0,0,0)), t1=now, since=floor(t0/1000), until=null; 'yesterday' — t0=начало вчерашних суток, t1=начало сегодняшних, since и until заданы (окно ровно вчера); number N — t0=now-N*DAY_MS, t1=now, since задан, until=null; 'all' — t0=min(rows.start) (пустой rows→t0=null как сигнал 'нет данных'), t1=now, since=null, until=null. Единый источник домена для renderGantt и since/until для fetchGitCommits.
