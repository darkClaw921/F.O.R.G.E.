# next-step-popup

Интерактивный hover-попап «Следующий шаг» (tmux-web/static/js/sessions/next-step-popup.js + css/next-step-popup.css). Singleton .next-step-popup на body, position:fixed. При наведении на .session-item.has-next-step показывает textarea с предложением (Отправить в терминал → POST /api/echo/next-steps/:session/send) и поле коррекции (Сохранить → .../feedback). Удерживается пока курсор над попапом (mouseenter cancelHide / mouseleave scheduleHide, HIDE_DELAY_MS=160). ВАЖНО про скролл: window scroll-обработчик (capture) скрывает попап ТОЛЬКО если scroll исходит ВНЕ попапа — скролл внутри textarea/тела попапа (popupEl.contains(e.target)) игнорируется, иначе длинный текст нельзя прочитать прокруткой. show() вызывает autoSizeText(): textarea предложения авто-растёт под содержимое (height=scrollHeight) в пределах CSS max-height:50vh, дальше скролл внутри. Попап: width 360px, max-height calc(100vh-24px), overflow-y auto; nsp-text min-height 92px.
