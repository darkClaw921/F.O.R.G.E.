# duration_until_next

daily_report::scheduler::duration_until_next(now: DateTime<Local>) -> Duration: чистая pub(crate) функция расчёта времени до ближайших локальных 23:00 (REPORT_HOUR=23). Если now строго раньше сегодняшних 23:00 — возвращает интервал до них; иначе (в 23:00 или позже, включая после полуночи) — до 23:00 следующего дня. DST-неоднозначности разрешаются через local_at (LocalResult: Single/Ambiguous→ранняя граница, None→+24h). Детерминированный hook для unit-тестов (6 тестов покрывают границы: до/в/после 23:00, после полуночи, инвариант (0,24h]).
