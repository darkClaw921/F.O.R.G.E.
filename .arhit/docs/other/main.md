# main

Точка входа devforge (tmux-web/src/main.rs). #[tokio::main] async fn main() -> anyhow::Result<()>. В самом начале main() (до tracing_subscriber/init и любых тяжёлых операций) обрабатывается лёгкий флаг --help / -h: std::env::args().any(|a| a == "--help" || a == "-h") печатает usage (имя бинаря, дефолтный порт 3000, ссылка на репо) и std::process::exit(0). Это нужно для Homebrew test do блока (assert_match "devforge", shell_output("#{bin}/devforge --help")) и не зависит от инициализации tokio runtime, тёплых TCP-сокетов, файловой системы, отсутствия tmux в PATH и т.д. После --help-handler идёт нормальный bootstrap: tracing_subscriber, ProjectStore::load, broadcast/watch каналы, AttentionState, NotifierLoop, регистрация всех axum-роутов (включая static_embed::serve_static как fallback) и axum::serve на SocketAddr 0.0.0.0:3000.
