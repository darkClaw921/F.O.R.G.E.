# pwa

pwa.rs — PWA HTTP-хендлеры + DTO. Edge-тесты в pwa.rs mod tests (17, DTO-слой): lenient serde — SubscribeReq игнорирует expirationTime (null/число) и contentEncoding (нет deny_unknown_fields, защита от тихой регрессии браузеров); отклонение тел без keys/endpoint/auth, битого JSON/пустого/null; unicode device_label + длинный endpoint без обрезки; UnsubscribeReq требует endpoint; PwaConfig camelCase; TestResp/OkResp контракт. Handler-уровневые тесты в main.rs mod tests (7, через test_app_state(Option<PwaCtx>)): opt-in 404-инвариант (get_pwa_config/post_push_subscribe/unsubscribe/test при pwa=None → 404); 200 с реальным VAPID public key при pwa=Some; subscribe ставит серверный created_at + дедуп по endpoint; идемпотентный unsubscribe; push_test пустой стор → {sent:0,pruned:0} без сети.
