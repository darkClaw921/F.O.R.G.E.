# TodoStore::update

TodoStore::update(&self, id, title: Option<String>, description: Option<Option<String>>, plan_mode: Option<bool>, auto_promote: Option<bool>) -> Result<Option<Todo>> (tmux-web/src/todos.rs). Частичное обновление существующей карточки по id (поиск по всем root-bucket'ам Inner.by_path). Семантика параметров: None=не трогать; для title Some(s)=записать; для description Some(None)=очистить, Some(Some(s))=записать; для plan_mode/auto_promote Some(b)=записать b. При успехе обновляет updated_at (now_rfc3339) и атомарно сохраняет файл (save_locked под write-lock). Возвращает обновлённую копию Todo или None если id не найден. Вызывается из patch_todo (PATCH /api/todos/:id, main.rs) — Фаза 1 передаёт auto_promote=None (Фаза 2 прокинет req.auto_promote). Параметр auto_promote добавлен для авто-промоута TODO по очереди.
