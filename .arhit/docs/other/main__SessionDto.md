# main::SessionDto

DTO в tmux-web/src/main.rs, сериализуемый в JSON для GET /api/sessions. Поля: name/attached/windows и прочее из tmux::SessionInfo через #[serde(flatten)] + needs_attention: bool + is_generating: bool + generating_since_secs: Option<u64> + folder_id/folder_label: Option<String> + origin: String. needs_attention: true когда в pane Claude prompt И сессия primary в дедупе. is_generating: true когда за тик watcher'а (1.5с) изменились последние 50 строк pane (gen_hash50 prev≠current) И сессия primary в дедупе по (session_group, gen_hash50). generating_since_secs (новое): длительность ТЕКУЩЕЙ непрерывной серии генерации в секундах из attention::generating_age_snapshot(); заполняется только если is_generating==true, иначе None (#[serde(skip_serializing_if=Option::is_none)]); чисто информационное поле для tooltip синего индикатора работы на фронте (sessions.js::generatingTooltip). folder_id/folder_label: папочная группировка __folder:<path> из cwd сессии. origin: всегда 'local' для локальных DTO.
