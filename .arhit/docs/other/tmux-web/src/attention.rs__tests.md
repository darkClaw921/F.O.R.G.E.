# tmux-web/src/attention.rs::tests

Тестовый модуль attention.rs. Включает: (1) детектор Claude-prompt тесты (PROMPT_FIXTURE, FILE_CREATE_FIXTURE, PLAN_FIXTURE, QUESTION_FIXTURE), (2) attention_state_snapshot_and_set / attention_state_is_cheaply_cloneable, (3) update_generation_* серия под новый API prev≠current (first_call_returns_false, same_hash_returns_false, different_hash_returns_true, does_not_write_to_generating_map), (4) dedup_attention_* для deduplicate_attention (helper mk_session), (5) dedup_generating_* для deduplicate_generating (helper mk_gen) — same_gen_hash_keeps_only_primary, different_hash_no_grouping, attached_wins_over_detached, same_group_unifies_even_with_different_hash, no_change_keeps_all_false. После Phase 4 удалены устаревшие sliding-window тесты (three_changes_still_false_window_not_full, four_distinct_changes_return_true, oscillation_returns_false, resets_when_pane_stops_changing, tracks_multiple_sessions_independently, single_change_returns_false, generating_snapshot_returns_independent_copy) — они оперировали удалённой константой GENERATION_WINDOW и не соответствуют новой семантике update_generation (changed = prev != current, без записи в карту generating). Всего 28 тестов в модуле.
