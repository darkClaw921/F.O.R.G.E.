# tmux-web/static/app.js::renameSession

async function renameSession(oldName, origin) — UI-обработчик кнопки 'rename' в session-item. Промптит пользователя для нового имени (window.prompt с pre-fill oldName); если пусто/Cancel/совпадает со старым — выходит. Шлёт PATCH /api/sessions/:name с body { name: <new> } через apiFetch (origin-aware, поддерживает remote-серверы). Сервер возвращает финальное имя (с tmux_prefix), считывает его из ответа (fallback на trimmed input). Если переименована state.currentSession — дисконнектит WS, показывает placeholder, обновляет список и переоткрывает сессию по новому имени через openSession(newName, origin). Иначе просто fetchSessions(). Ошибки показывает через window.alert. Кнопка btn-rename размещена в session-actions рядом с btn-kill.
