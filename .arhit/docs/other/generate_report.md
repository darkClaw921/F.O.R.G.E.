# generate_report

Генерирует «Сводку дня» за указанную дату (NaiveDate) и upsert'ит её в daily_reports под ключом YYYY-MM-DD. Собирает контекст дня (Echo-чаты, tmux-панели, git-активность). Если все три источника пусты — пишет маркер NO_ACTIVITY_RU без вызова Claude. Иначе строит prompt из мета-prompt'а и блоков данных, прогоняет через state.runner.one_shot. Эффективный промпт отчёта читается из app_settings по ключу PROMPT_KEY_REPORT (daily_report.report_prompt): пользовательский оверрайд используется только если он непустой (filter по trim), иначе фолбэк на дефолтную константу REPORT_META_PROMPT. Отдельным проходом вызывает generate_suggestions для предложений задач (деградирует до пустого массива при ошибках). Возвращает DailyReport со стабильным id благодаря upsert.
