# capture_now

session_history::capture_now(store: &HistoryStore) async — единая точка снятия снимка истории tmux-сессий. Опрашивает tmux::list_sessions(), для каждой сессии вызывает tmux::list_windows() (при ошибке — пустой список окон с tracing::warn), собирает Vec<(SessionInfo, Vec<WindowInfo>)> и передаёт в store.snapshot(), который upsert'ит записи по ключу name+\0+path (обновляя last_seen/windows/folder_label у существующих, создавая first_seen=last_seen=now у новых) и атомарно персистит JSON на диск (~/.config/forge/session_history.json). Переиспользуется в двух местах main.rs: (1) периодическим snapshot-воркером (tokio::spawn, tokio::time::interval 3600s, срабатывает немедленно на первом тике) и (2) graceful-shutdown хуком внутри with_graceful_shutdown — после wait_for_shutdown_signal() и forge_echo::shutdown() вызывается capture_now(&history_for_shutdown) чтобы зафиксировать финальный снимок перед остановкой процесса. Зависимости: crate::tmux, crate::session_history::HistoryStore. Не возвращает значения; все ошибки логируются, не паникуют.
