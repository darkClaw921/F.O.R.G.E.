# push_store

PushSubscriptionStore — персистентное opt-in хранилище Web Push подписок (~/.forge/push_subscriptions.json, Arc<RwLock<Inner>>, atomic save tmp+rename, дедуп по endpoint). Edge-тесты (#[cfg(test)] mod tests, 32 теста): ротация всех keys (p256dh+auth)+label при upsert; сохранение порядка вставки и in-place обновление; remove несуществующего на свежем сторе не создаёт файл (lazy creation); prune empty/nonmatching не трогают диск, prune all пишет валидный []; prune дедуп повторов в срезе; roundtrip device_label=Some с unicode; битый JSON/объект-вместо-массива/missing-field/пустой файл/[] → пустой список без паники; created_at verbatim; диск всегда валиден между мутациями; create_dir_all для parent; pretty-JSON формат; конкурентные upsert distinct/same + churn upsert/remove под RwLock без потерь/паники; list() независимый снимок; clone шарит Arc-состояние; upsert Err при недоступной записи без паники; default_subscriptions_path с/без HOME.
