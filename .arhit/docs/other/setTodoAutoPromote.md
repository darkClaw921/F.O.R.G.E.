# setTodoAutoPromote

tmux-web/static/js/tasks/crud.js — CRUD-функция переключения флага авто-промоута у TODO-карточки. Сигнатура: setTodoAutoPromote(id, value) -> Promise<bool>. Находит todo в state.todosData по id; если не найден или значение уже равно next — выходит (true). Оптимистично проставляет todo.auto_promote = !!value и вызывает renderTasks() для немедленной перерисовки борда, затем шлёт PATCH /api/todos/:id с телом { auto_promote: next } через apiFetch(url, opts, origin), где origin = dtoOrigin(todo) || 'local' (по образцу promoteTodo/deleteTodoLocal — todos живут в плоском массиве state.todosData, не в state.tasksData.issues, поэтому применяется ручной оптимистичный апдейт+rollback, а не applyOptimisticPatch). При не-ok ответе или исключении: откатывает todo.auto_promote к prevValue, ре-рендерит, показывает window.alert и возвращает false. Сервер на успешный PATCH рассылает WS Upsert (ws_todos.rs), который приводит состояние к авторитетному. Вызывается из renderTodoCard (render.js) по клику на .auto-promote-toggle.
