# static_embed

static_embed::serve_static — axum-handler embedded-static (rust-embed). Edge-тесты (#[cfg(test)] mod tests, #[tokio::test], 11 тестов): sw.js и manifest.webmanifest → Cache-Control: no-cache (точное == path, не contains); app.js/style.css/index.html/icon.png → БЕЗ Cache-Control; mime через mime_guess 2.0.5 (js→text/javascript, css→text/css, png→image/png, html→text/html); 404 для отсутствующего пути с телом '404 Not Found'; '/' → index.html; trailing-slash → <dir>/index.html (404 для несуществующего); query-string не ломает резолюцию имени (/sw.js?v=123); вложенный js/sw.js != строгий 'sw.js'; идемпотентность повторных вызовов. Заголовки читаются ДО into_body(); body через axum::body::to_bytes.
