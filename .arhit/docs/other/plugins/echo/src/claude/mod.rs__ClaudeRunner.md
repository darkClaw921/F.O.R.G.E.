# plugins/echo/src/claude/mod.rs::ClaudeRunner

Фасад над Claude CLI с поддержкой stream/one_shot/cancel. Поля: cli_path (PathBuf), semaphore (Arc<Semaphore> с N permits), running (Arc<Mutex<HashMap<RunId, AbortHandle>>>). new(cli_path, max_parallel) — НЕ падает если cli не существует, только warn-log (важно для тестов и dev-машин без Claude CLI). stream(req) → mpsc::Receiver<ClaudeEvent>: спавнит claude -p --output-format stream-json --include-partial-messages --verbose [--model] [--append-system-prompt] <prompt>, читает stdout построчно через BufReader::lines, парсит через events::parse_line, шлёт в канал; stderr идёт в tracing::debug. one_shot(req) → RunResult: собирает все TextDelta в text + финальный usage из Result-event. cancel(run_id) → bool: aborts spawned task; kill_on_drop=true автоматически убьёт child процесс.
