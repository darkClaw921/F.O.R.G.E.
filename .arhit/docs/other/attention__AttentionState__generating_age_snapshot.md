# attention::AttentionState::generating_age_snapshot

Метод AttentionState (tmux-web/src/attention.rs): возвращает HashMap<session_name, u64> — длительность текущей непрерывной серии генерации в секундах для каждой сессии, у которой is_generating сейчас активен. Источник — поле gen_started_at: Arc<RwLock<HashMap<String, Instant>>>, которое set_generating заводит при фронте флага false→true (Instant::now()), держит пока флаг true (true→true не сбрасывает отметку), и удаляет при →false. watcher_loop дополнительно прунит gen_started_at в cleanup-фазе для исчезнувших сессий. Используется хендлером GET /api/sessions для заполнения SessionDto::generating_since_secs, на основе которого фронтенд (sessions.js::generatingTooltip) строит tooltip синего индикатора работы ✶: «в сессии X содержимое терминала меняется — генерация идёт уже N с». На дедуп/детект логику не влияет, поле чисто информационное.
