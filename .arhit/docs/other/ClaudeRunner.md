# ClaudeRunner

plugins/echo/src/claude/mod.rs. Фасад над Claude CLI (spawn/stream/cancel), cheap-clonable через Arc. Поля: cli_path, semaphore (ограничивает max_parallel одновременных run), running (HashMap run_id->AbortHandle), run_timeout (Duration, ZERO=отключён). new(cli, max_parallel) ставит run_timeout=DEFAULT_RUN_TIMEOUT_SECS(300). with_run_timeout(secs) переопределяет из EchoConfig.run_timeout_secs (вызывается в init_with_config). stream(): permit берётся внутри spawned-таски и держится весь run; read-loop чтения stdout обёрнут в tokio::time::timeout(run_timeout) — по таймауту child.kill()+ClaudeEvent::Error, что освобождает permit. Это закрывает разом все пути run'а (чат, autonomous, next_step, memory), т.к. все идут через stream(). Без таймаута зависший CLI выедал бы все слоты семафора.
