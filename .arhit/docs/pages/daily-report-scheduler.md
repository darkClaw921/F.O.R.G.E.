Phase 3 фичи «Сводка дня» (epic forge-9db). Добавлен модуль plugins/echo/src/daily_report/scheduler.rs — фоновый воркер авто-генерации дневной сводки в локальные 23:00. Содержит spawn() (loop с tokio::select! sleep vs state.shutdown.cancelled()), pub(crate) duration_until_next(now) для тестируемого расчёта интервала, и приватный local_at() для DST-safe конструирования локального DateTime. Модуль объявлен через 'pub mod scheduler' в daily_report/mod.rs. Воркер зарегистрирован третьим в lib.rs::spawn_workers (state.register_worker) рядом с scheduler::spawn (autonomous) и memory::scheduler::spawn (UTC rollover); tracing::info лог обновлён. В отличие от memory::scheduler (UTC, раз в час, маркер в БД) этот привязан к локальному времени пользователя. 6 unit-тестов на расчёт интервала.
