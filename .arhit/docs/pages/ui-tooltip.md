Кастомный tooltip (tmux-web/static/js/ui/tooltip.js) — замена нативному title. Singleton <div class='forge-tooltip'> на body (стили в base.css, z-index 100001 выше всех overlay). Показывается для любого элемента с атрибутом data-tooltip='<текст>'. Логика навешана через ДЕЛЕГИРОВАНИЕ на document (mouseover/mouseout/focusin/focusout + scroll/resize→hide) — поэтому работает для динамически перерисовываемых элементов (sidebar пересобирается каждые 3с) без переподписки. position: fixed, над целью по центру с зажимом в viewport (fallback вниз если не влезает сверху). Текст читается из data-tooltip в момент показа → всегда актуальный. Экспортирует initTooltips() (идемпотентна), вызывается из core/bootstrap.js. Первый потребитель — синий индикатор работы ✶ (.claude-spark в sessions.js::buildSessionItem): spark.dataset.tooltip = generatingTooltip(s).
