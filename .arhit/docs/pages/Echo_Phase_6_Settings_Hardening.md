Phase 6 features for Echo plugin: 1) EchoConfig (plugins/echo/src/config.rs) — TOML + env loader. 2) UserSettings extension (tmux-web/src/user_settings.rs): echo_default_model: Option<String> и echo_notifications_enabled: bool. 3) Settings modal Echo tab (tmux-web/static/js/echo/settings.js + settings/modal.js). 4) Rate-limit 30/min per WS user_message (sliding-window VecDeque<Instant> в ws/mod.rs). 5) Autonomous daily token cap (state.config.autonomous_max_tokens_per_day, агрегация SUM(tokens_in+tokens_out) FROM task_runs WHERE started_at>=utc_today_start). При cap reached: task disabled + warn Notification. 6) Graceful shutdown — CancellationToken в EchoState, forge_echo::shutdown() вызывается при SIGTERM. ClaudeRunner.shutdown() abort'ит активные runs, kill_on_drop убивает children. 7) Tracing namespace forge_echo (FORGE_ECHO_DEBUG=1 включает debug-уровень). 8) +14 integration tests (plugins/echo/tests/phase6_repos_and_parsers.rs) + 5 smoke tests (tmux-web/tests/echo_smoke.rs). Total: ~167 tests up from 131 baseline.
