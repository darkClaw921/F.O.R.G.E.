Раскладка прокрутки и оси гант-диаграммы (tmux-web/static/js/tasks/gantt.js + tmux-web/static/css/tasks.css). Структура DOM: #tasks-gantt (flex column, height:34%, overflow:hidden — тулбар зафиксирован сверху) -> #gantt-toolbar (flex 0 0 auto) + #gantt-canvas (position:relative, flex:1, min-height:0, overflow-y:auto — СКРОЛЛ-КОНТЕЙНЕР). Внутри #gantt-canvas: .gantt-axis (прямой потомок, position:sticky; top:0; z-index:5; background:var(--bg) — прилипает к верху при прокрутке, непрозрачный фон не даёт строкам/чертам просвечивать), затем .gantt-rows (position:relative). В .gantt-rows: все .gantt-row, заглушка .gantt-empty и overlay .gantt-commits-overlay (absolute top/bottom/left/right:0). overlay внутри .gantt-rows покрывает полную высоту всех строк -> вертикальные черты .gantt-commit доходят до самого низа при прокрутке (раньше overlay был в #gantt-canvas с bottom:0 и обрезался по видимой высоте). Ось в дневных диапазонах ('today'/'yesterday', span<=2*DAY_MS) показывает время HH:MM (fmtAxisTime), иначе даты (fmtAxisDate).
